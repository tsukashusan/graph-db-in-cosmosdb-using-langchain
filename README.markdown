# graph-db-in-cosmosdb-using-langchain

## 準備

### Python 仮想環境の作成
```
python -m virtualenv -p  python3.12 venv
```

### PowerShell ポリシー変更
```
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Bypass -Force
```

### アクティベート
```
.\venv\Scripts\activate.ps1
```

### パッケージのインストール
```
pip install -r requirements.txt
```
